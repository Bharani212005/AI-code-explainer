# 🧠 Code Explainer App (Streamlit + Gemini AI)

## 🚀 Overview

This project is a **Code Explainer Web App** built using **Streamlit** and **Google Gemini AI**.
It allows users to input any code or question and get a **clear explanation of the code** instantly.

---

## ✨ Features

* Explain any code in simple terms
* Fast responses using Gemini 2.5 Flash
* Clean and minimal Streamlit UI
* Helpful for students and beginners

---

## 🛠️ Tech Stack

* Python 🐍
* Streamlit 🎈
* Google Generative AI (Gemini API)

## ⚙️ Installation

1. Clone the repository:

```bash
git clone https://github.com/your-username/code-explainer.git
cd code-explainer
```

2. Install dependencies:

```bash
pip install streamlit google-generativeai
```

3. Set your API key:
   Replace the API key in your code:

```python
genai.configure(api_key="YOUR_API_KEY")
```

---

## ▶️ Usage

Run the app:

```bash
streamlit run app.py
```

### Steps:

1. Enter your code or question
2. Click **Submit**
3. Get a detailed explanation instantly
